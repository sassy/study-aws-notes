# CloudFormation
* AWSのシステムの構成を管理をするツール
* インフラストラクチャ管理を簡略化
* インフラストラクチャをすばやく複製
* インフラストラクチャの制御や変更の追跡も簡単

## AWS CloudFormation テンプレート
* JSON/YAML で記述

```JSON
{
    "AWSTemplateFormatVersion": "2019-09-09",
    "Description": "Sample",
    "Resources": {
        "(リソースを識別するための名前)" : {
            "Type": "(AWSのサービスのタイプ)"
            ...(中略)...
        }
    }
}
```

## スタック
* リソースはスタックと呼ばれる単一のユニットとして管理
* スタックを作成、更新、削除することで、リソースのコレクションを作成、更新、削除
